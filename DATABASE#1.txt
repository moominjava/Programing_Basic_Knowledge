오라클 대용량 데이터 저장 타입
CLOB -  사이즈가 큰 데이터를 외부 파일로 저장하기 위한  데이터 타입

BLOB - 바이너리 데이터를 DB 외부에 저장하기 위한 타입
        - 즉 컴퓨터가 인식하는 모든 파일(이진데이터)을 저장하는 타입
        - 최대 4GB

이진 문자열
RAW - 오라클에 텍스트 뿐만 아니라 이진 데이터도 저장가능
       - 디스크에 저장되는 측면에서 VARCHAR2와 비슷함(가변길이)
       - 최대 2000바이트

데이터 베이스 문자 세트
CHARACTERSET - 데이터베이스에서 취급하는 문자 코드
ex)  a -> 문자 저장 -> 이진값 -> 어떤 바이너리값으로 변환할지(문자코드, 문자집합)

★데이터베이스 설정에 지정하는 문자 코드는 데이터 베이스를 가동시키는 
서버OS의 문자를 사용하는 경우가 많다. 
유명한 곳 : AL32UTF8, JA16SJISTILDE, JA16EUCTILDE
